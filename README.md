## 1 | INTRODUCCIÓN

**PAKETS** es un programa de gestion de paquetería (de ahí el nombre) que permite controlar las entradas y salidas de los mismos tanto por libre como conectado a través de API's de las diferentes agencias de reparto como Correos, Seur, Impost, MRV, DHL, Tipsa, Ctt... y homónimos de estas.

### FUNDAMENTO
PAKETS tiene la tarea de **Simplificar** y **Centralizar** toda la logística en un solo lugar, a diferencia de recurrir a cada solucion de cada agencia que se resume en perdida de tiempo y un completo lio, en especial para aquellos que trabajan con más de un transportísta.

## 2 | CARACTERÍSTICAS PRINCIPALES
- COMUNICACION CON API'S: Gracias al uso de estas, cada persona puede hacer de PAKETS el programa ideal para comunicarse con solo las agencias que usted tenga, haciendo de este un programa a la medida.

- COMPLEMENTOS: Estos permiten que PAKETS se pueda adaptar a cualquier ámbito de trabajo, pudiendo usar los disponibles en la comunidad o crear los suyos propios

- TRABAJO SIN INTERNET: Una de las principales ventajas que ofrece PAKETS frente a los otros sistemas es la **independencia de internet**, solo es necesario para la comunicación con las API's de las agencias con las que trabaje, lo que permite que en caso de no disponer de acceso a la red, uno pueda seguir trabajando sin molestias.

- COMUNICACIÓN CON EQUIPOS LOCACES SIN NECESIDAD DE SERVIDORES: PAKETS puede comunicarse con otros equipos dentro de la misma red de area local (LAN) a la que esten conectados sin la necesidad de depender de servidores costosos, ello se traduce en multiples posibilidades como:
    - Trabajo simultaneo entre 2 o mas equipos donde PAKETS esté instalado.
    - Comunicacion con otros programas (ver compatibilidad).
    - Comunicación con complementos externos o instalados en el propio equipo.

- CONTROL DEL CONTENIDO: Además de la gestion de logística, PAKETS permite añadir manualmente el contenido del paquete y asignarlo al número de seguimiento proporcionado por la agencia. Esto permite mantener un mayor control del inventario junto con diversas herramientas que facilitan la tarea, todo ello manteniendo la confidencialidad del contenido de los paquetes puesto que estos datos no se comparten con la agencia.

## 3 | INSTALACIÓN Y REQUISITOS
PAKETS es un programa que demanda bajos recursos y es compatible con la mayoría de equipos, he aquí una lista de los requisitos mínimos para su instalación:
> [!WARNING]
> **PAKETS solo es compatible con Microsoft Windows 7(SP1), Windows 8.1, Windows 10 y Windows 11.**

| | REQUISITOS MÍNIMOS | REQUISITOS ÓPTIMOS |
|---|---|---|
|CPU| Intel® Core i3 / AMD® Ryzen 3 | Intel® Core i3 / AMD® Ryzen 3 |
|RAM| 4GB (8 si se desean instalar complementos, ver especificaciones) | 8GB (16 si se desean instalar complementos, ver especificaciones) |
|GPU| Gráficos integrados Intel / AMD Radeon |
|CONEXIÓN A INTERNET| Solo LAN (Necesita conexion en caso de uso de API's o complementos que lo requieran)|
|ALMACENAMIENTO| 1 GB de espacio (Necesita más si se instalan complementos)
|TECNOLOGÍAS NECESARIAS| .NET Framework 4.8



La instalación de PAKETS es como la de cualquier otro programa:
- Descargue la version mas reciente de PAKETS o en su defecto, la versión compatible con su sistema
- Abra el instalador y siga las instrucciones que se muestren.

> [!TIP]
> Si va a actualizar el programa o a reinstalarlo, le recomendamos hacer una copia de los archivos para evitar cualquier problema en caso de que la actualización o instalacion falle.
